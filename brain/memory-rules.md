# Memory Rules — How the Brain Stays Alive

1. **2-hour rule.** Every investor interaction (meeting, call, email thread) becomes
   a `meetings/` page within 2 hours, while memory is fresh.
2. **Verbatim questions.** Investor questions are captured word-for-word into
   `questions/`. Paraphrasing loses the signal of what they actually fear.
3. **One objection = one page.** Each objection gets its own page with the
   3-layer ladder (acknowledge → counter-evidence → reframe as reason to invest).
4. **Commitments are sacred.** Anything promised ("we'll send X by Friday") goes to
   `commitments/` with a deadline. An overdue commitment is a fire.
5. **Stale check weekly.** During `/fundraising-retro`, mark pages untouched 30+ days
   as `status: stale` and decide: refresh or close.
6. **Gap analysis before every meeting.** The prep brief must contain a section
   "What we do not know" — missing intel is stated, never papered over.
7. **The brain feeds everything.** Deck answers, memo content, playbooks, and QA
   attacks are all generated FROM these pages, never from memory or vibes.
