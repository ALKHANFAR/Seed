# Financial Gates

- Actuals come from the accounting system / bank, never from memory.
- Projections live in clearly marked PROJECTED columns; assumptions listed beside them.
- Gross margin must include voice/LLM/telephony unit costs; show the per-minute math.
- CAC includes salaries of selling humans, not just ad spend.
- Burn = average of last 3 months actual, not "planned burn".
- Valuation ask must reference a logic (comps, dilution norms, milestone value) —
  "we feel it's worth X" fails the gate.
