# Traction Gates

| Claim type | VERIFIED requires |
|---|---|
| Revenue (MRR/ARR) | Bank/Stripe/invoice reconciliation, not CRM pipeline |
| Growth rate | Same metric, same definition, ≥3 consecutive periods |
| Customers count | Active + paying; trials and POCs labeled separately |
| Retention / churn | Cohort table exported, definition stated (logo vs revenue) |
| Usage (calls/minutes) | Dashboard export with date range |
| Logos | Written permission to name them, or "a leading <sector> company" |
| Case study results | Customer-confirmed numbers (email or signed one-pager) |

Vanity metrics (signups, waitlists, demo requests) are PARTIAL at best and never
lead a slide.
