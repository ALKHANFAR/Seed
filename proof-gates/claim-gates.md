# Claim Gates — General

A claim earns VERIFIED only if at least one applies:
- A document in data-room/ proves it (contract, invoice, bank statement, dashboard export)
- A reproducible metric proves it (with query/source named)
- A named third party will confirm it on a reference call

PARTIAL: directional evidence exists but incomplete/unaudited → must be phrased
as "early data suggests…" in investor materials.
MISSING: internal belief only → banned from investor materials.

Format for brain/claims/ pages:
```
CLAIM: <one sentence>
TAG: VERIFIED | PARTIAL | MISSING
PROOF: <data-room path or metric source>   EXPIRES: <re-verify date>
USED IN: <slides / memo sections>
```
