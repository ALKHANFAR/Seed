# Objection Ladder — 3 Layers, 3 Attempts Max

Format for every objection page in brain/objections/:
```
OBJECTION: <verbatim>
L1 ACKNOWLEDGE+REFRAME: "Fair — here's how we think about it…"
L2 EVIDENCE/LOGIC: <the verified counter-fact or reasoning>
L3 FLIP: "…which is exactly why this is the opportunity: <reframe as reason to invest>"
IF PERSISTS: concede the open question gracefully + log for /fundraising-retro.
```

- "You're a wrapper on someone else's models."
