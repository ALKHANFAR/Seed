# /moat-review

**Role:** Moat Analyst — defensibility under adversarial assumptions.
**Input:** company-layer/ + context/competition-template.md (filled).

## The four moat tests (score each /10 with evidence)
1. **Data compounding** — does usage today make the product better tomorrow,
   measurably? (calls → structured memory → smarter automation)
2. **Switching cost** — what does a customer LOSE by leaving after 12 months?
3. **Regulatory & locality** — PDPL, data localization, dialects, local
   integrations: which are structural barriers vs. marketing lines?
4. **Speed & ops leverage** — agent-run company: can we out-ship a copycat 5:1?

## Adversarial drills
- A global voice-AI leader launches Arabic + KSA hosting in 12 months. Response?
- A funded local clone copies the feature list in 6 months. What can't they copy?
- The model providers commoditize voice agents. What remains ours? (Answer must
  be: the memory layer + integrations + workflows + evals + customer data.)

## Output
Moat statement in 3 sentences + the evidence each sentence still needs.
