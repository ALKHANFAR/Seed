# /fundraising-retro   (weekly, 45 minutes, non-negotiable)

**Input:** crm/pipeline.md + brain/ pages updated this week + readiness score.

## Agenda
1. **Pipeline math:** meetings held / scheduled / gone cold. Conversion between stages.
2. **Question patterns:** what did 2+ investors ask this week? → update the deck or
   FAQ so next week's meetings pre-empt it.
3. **Objection patterns:** which objection repeated? → strengthen its 3-layer answer
   or fix the underlying reality (sometimes the objection is simply true).
4. **Proof delta:** which claims moved MISSING→PARTIAL→VERIFIED this week?
5. **Commitments audit:** anything overdue to any investor? (fire if yes)
6. **Readiness score:** update readiness/term-sheet-readiness-score.md. Two weeks
   with no movement = thesis problem, run /investor-office-hours again.
7. **One decision:** the single biggest change for next week. Write it down.

Output: a dated file in `retros/`.
